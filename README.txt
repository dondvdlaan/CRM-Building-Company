CRM-Building-Company

This project is based on the requirements of a building company of wooden log homes.
It concerns mainly a CRUD application for customers, projects, references, suppliers and
staff. Focus is on the projects section, while a project goes through a funnel mechanism from 
discovery, to offering, closed won/lost, contract and execution.

Summary:
- Back-end  : JAVA 17/Spring/Security/jpa
- Database  : MySQL
- Front-end : REACT
- CSS       : Material UI
- Testing   : JUNIT, Spring @MockBean, MockMvc and Mokito
